# Snakes
ကိုယ်သိချင်တဲ့မြွေကိုရွေးမယ်။ ကိုယ်ရွေးလိုက်တဲ့ မြွေရဲ့အကြောင်း၊ အဆိပ်ရှိမရှိ၊ အန္တရာယ်ရှိမရှိ စတဲ့အကြောင်းလေးတွေ ဖတ်ကြမယ်။

သဘာဝတွင်တွေ့ရှိရသော မြွေများကို ပုံနှင့်တကွ ပြသထားပါသည်။
မြွေပုံကိုနှိပ်၍ ထိုမြွေနှင့်ပတ်သက်သော အကြောင်းအရာ၊ အဆိပ်ရှိမရှိ၊ အန္တရာယ်ရှိမရှိ ကိုဖတ်ရှုနိုင်ပါသည်။
